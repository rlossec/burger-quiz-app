# Layout global + Login

---

## Layout global

Toutes les pages (hors Login) partagent une structure commune :

```
+------------------------------------------------------------------+
|  [Logo] Burger Quiz    Navbar : Burger Quiz | Questions | ...  [User] [Logout]  |
+------------------------------------------------------------------+
|                                                                  |
|  Fil d'Ariane (optionnel)  ou  Titre de la page                 |
|                                                                  |
|  +------------------------------------------------------------+  |
|  |                                                            |  |
|  |                    Zone de contenu                         |  |
|  |                    (liste, formulaire, détail)             |  |
|  |                                                            |  |
|  +------------------------------------------------------------+  |
|                                                                  |
+------------------------------------------------------------------+
```
